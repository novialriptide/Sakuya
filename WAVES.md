# SakuyaEngine .wave Documentation
## Commands
```
spawn <entity_key> <spawnpoint_key> <spawn_animation_key>
    Spawn an entity at a spawn point.

    Example:
        spawn 4 2 0
        spawn 0 6 0
        spawn 9 0 1

wait <milliseconds>
    Time to wait to resume commands

    Example:
        wait 3000
        wait 1500
```
