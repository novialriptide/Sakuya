# Sakuya
A game engine, named after a Touhou character, specifically made for bullet hells. This engine depends on [pygame](https://www.pygame.org/)

## Features
1. Primitive Physics Engine
2. Bossbar
3. Circles
4. Text
5. Time

## Planned Features
1. Tilemaps
2. Replays
3. Buttons
4. Entity Animations
5. World Save
